Projects

  • Lean 6 Sigma Green Belt Project

    Completed a Lean Six Sigma Green Belt project on the reception center medical screening process at San Quentin State Prison improving compliance from 2% and a baseline capability aggregate of 29.96% compliance to 84% during the transition phase of the project through application of the lean 6 methodologies and tools. Scope of project affects 6 total state prisons with 3 more to be positively impacted by state wide changes resulting from this project

  • Runes.sh – Elder Futhark Runes In The Terminal

  • Runes

    Language(s): – ( R - Markdown - yaml )

    An R package that uses regular expressions to convert alpha characters to Elder Futhark Runes. This package includes over 50 unit tests, integrated Travis CI, uses semantic versioning in a comprehensive changelog, and is actively developed on github. This package has been published to CRAN the Comprehensive R Archive Network, with potential applications in linguistics.

  • Flash.sh – flashcards in your terminal!

    Language(s): – ( Bash )

    A script that reads a 3 column CSV file and shuffles the entries around so you can practice what ever you'd like with flashcards, all in the comfort of your terminal.
